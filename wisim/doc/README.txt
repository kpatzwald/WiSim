System Requirements
-------------------

MySQL 3.x or higher
Java 2 Runtime Environment 1.4.x or higher

Installation
------------
Unpack the *.zip-File in a directory of your choise.

Start
-----

Windows
-------
Just doubleclick the wisim.jar or run "java -jar wisim.jar" in a command. Make sure, you have enough privileges to create a new database or ask your system administrator.

Linux / Unix
------------
Run "java -jar wisim.jar" in a shell. Make sure, you have enough privileges to create a new database or ask your system administrator.

COMPILING THE SOURCECODE:
-------------------------
There is a build.xml in the src directory. Open it with a texteditor and change the root property to the directory you have installed wisim. If you have installed ANT (http://ant.apache.org) go to the src directory und run "ant".
